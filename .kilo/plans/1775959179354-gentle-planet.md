## Phase 6: Animations (Framer Motion)

### 6.1 Scroll Reveal Hook
```typescript
// lib/animations.ts
'use client';
import { motion, useInView } from 'framer-motion';
import { useRef } from 'react';

export function useScrollReveal() {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true, margin: '-100px' });
  return { ref, isInView };
}
```

### 6.2 Apply to Sections
- Hero: Gradient animation + floating icons
- Problem cards: Stagger reveal (0.1s delay each)
- Feature cards: Hover lift + scale
- Stats: Counter animation on scroll
- FAQ: Accordion expand
