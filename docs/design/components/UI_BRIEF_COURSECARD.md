**Task Title:** Design Premium Course Card UI

**Task Description:** 
Please design a premium, high-converting "Course Card" for the Upskiill platform. The layout needs to be clean but information-dense. Since there is a lot of data, please use creative hover states (like sliding up the description or buttons) to keep the default view uncluttered. 

**What is a Course Card? (Do not confuse with the Course Landing Page!)**
A "Course Card" is a small, compact, rectangular tile (usually something like 300x400 pixels) that acts as a summary/preview. These cards are displayed in grids or lists alongside many other cards on the Homepage or Search Results pages so users can browse multiple courses at once. It is **NOT** the full, full-screen course description page. Think of it like a movie poster tile on Netflix or a product card on Amazon.

**Color & Style Instructions:**
*   **Primary Action:** Deep Blue (`#3D5AFE`) for the main buttons and active states.
*   **Card Body:** Crisp White (`#FFFFFF`) background with very subtle light-grey borders (`#E2E8F0`).
*   **Typography:** Dark Navy (`#1F2A44`) for bold titles and prices. Slate Grey (`#64748B`) for secondary text.
*   **Accents:** Golden Amber for Star Ratings, Emerald Green for "Free" labels and Progress bars, and Rose Red for Discounts/Alerts.
*   **Interactions:** The card must feel alive. It should elevate with a soft drop-shadow when hovered over.

**Items to Include in the Design (Checklist):**
*   Course Thumbnail (image)
*   Badge (Bestseller / New / Top Rated)
*   Course Title
*   Instructor Name
*   Instructor Avatar (optional)
*   Rating (⭐ + number)
*   Number of Students
*   Course Duration
*   Number of Lessons (optional)
*   Skill Level (Beginner / Intermediate / Advanced)
*   Course Category (optional)
*   Short Description (hover or expanded)
*   Price (current)
*   Old Price (strikethrough)
*   Discount Label (e.g., “50% off”)
*   “Free” Label (if applicable)
*   Call-to-Action Button (View Course / Enroll Now)
*   Add to Cart / Wishlist Icon (optional)
*   Progress Bar (if user enrolled)
*   “Updated Recently” Label (optional)
